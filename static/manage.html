Add Token

1. Go to labs.google.com/fx and log in
2. Press F12 → Application tab → Cookies → Copy __Secure-next-auth.session-token
3. Paste here. Auto-converts to Access Token on save.

If you already have a Project ID, enter it directly,Leave empty to create a new project

Overrides browser captcha proxy for current token only; leave empty to use global captcha proxy

Edit Token

1. Go to labs.google.com/fx and log in
2. Press F12 → Application tab → Cookies → Copy __Secure-next-auth.session-token
3. Paste here. Auto-converts to Access Token on save.

Changing Project ID updates the project used by this token

If filled, overrides global captcha proxy; clear to revert to global proxy
